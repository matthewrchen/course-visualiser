# Course Visualiser
This project is an interactive curriculum visualiser for the Bachelor of Engineering (Honours) degree offered by the University of Auckland, built on the Next.js framework.

## 📌 Overview
Planning what electives to take can be intimidating due to a lack of clear structure and can be annoying to deal with since prerequisites are scattered on different course pages.

Course Visualiser solves that problem by mapping the progression of all courses available for study, from Part I to IV.

### 🌟 Features:
- **All 10 specialisations supported:**
    - Biomedical Engineering
    - Chemical and Materials Engineering
    - Civil Engineering
    - Computer Systems Engineering
    - Electrical and Electronic Engineering
    - Engineering Science
    - Mechanical Engineering
    - Mechatronics Engineering
    - Software Engineering
    - Structural Engineering
- **Interactive sidebar:** Click any course node to instantly view its full course prescription, prerequisites, and restrictions.

## 🛠️ Technologies used
- **Next.js**
- **React**
- **Tailwind CSS**
- **Cytoscape.js**
